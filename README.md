# OpenGL Tutorial - An Introduction on OpenGL with 2D Graphics

An Introduction on OpenGL with 2D Graphics article.

## Setup

Install all dependencies
```
pnpm install
```

Start the website
```
pnpm dev
```

Build the website
```
pnpm build
```

Preview the built website
```
pnpm preview
```

## Why use pnpm?
1. Save disk space
2. Boost installation speed
